Application information

  • Technology Used:
  • Application is built with HTML, CSS and Javascript in the NodeJS environment with a Mongodb database.
  • The application uses Express and the spoonacular API to provide the recipe information.
  • JWT web tokens used for authentication.
  • User Functionality:
  • Random recipes shown on home page. Users can search for recipes based on a search term.
  • Users can look for recipes based on a search term.
  • Users can create a profile by registering with an email and password. This allows users to save a list of recipes to the database.
